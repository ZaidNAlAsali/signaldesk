# Security policy

## Supported versions

SignalDesk is a portfolio project. Security fixes are applied to the latest `main` branch only.

## Reporting a vulnerability

Please do not open a public issue containing exploit details, credentials, or personal data. Report the issue privately through GitHub's security-advisory feature on this repository. Include the affected component, reproduction steps using synthetic data, and expected impact.

Do not test against systems or accounts you do not own. The project has no public production deployment.

## Scope reminder

SignalDesk has no authentication or authorization and is not intended for exposure to untrusted networks. See [`docs/safety.md`](docs/safety.md) for the threat model and required controls before real deployment.
